Introduction

Ever wondered how the websites you browse every day are made? Well, the answer is typically web developers. Web developers are responsible for every aspect of the web. From coding interactive features to designing the website, these people do it all.

Payscale

Entry level pay typically starts at around $50,500. Advanced positions can pay up to $80,000.. However, it can be more or less depending on reliability of the employer or if you are self-employed.

Pros and cons


Pro Con

Careers are flexable and positions range from specializing in one thing to everything. This means you can get into it regardless of what you specialize in. May be hard for some to get started due to all of the different tools used.
Good pay in advanced positions. Some of the lower level positions don't pay that well compared to other low-level tech jobs.

Types of web developers

There are three main types of web developers. Frontend, backend, and fullstack. Each one is responsible for their own parts of the website's overall functioning and rely on each other. Below are the various web developers, along with their roles and responsibilities.

Frontend

Frontend web developers are responsible for the parts of the website that you (the user) see. They design the layout, create the elements, and build the interactive portions of the website.

Backend

Backend web developers are responsible for the parts of the website that you don't see. They implement features such as the server and databases. Backend web developers are the reason we have search features, accounts, online stores, and much more. Without them, websites would only have a small portion of interaction.

Fullstack

Fullstack web developers do it all. They are experts in both fields of web development. They are also typically quite knowledgeable in hosting, server management platforms, as well as business parts of the website.

Programming languages

Frontend

Frontend web developers work in three main languages. HTML, which is responsible for the barebones elements, CSS, which is responsible for the design and colors, and Javascript, which does all of the interactive things.
The best way to explain it is using a sentence. for example, "The black cat pounced."
The cat is the element, black is the design, and pounce is the interaction.

Backend

Backend web developers use a lot more complex tools. They need powerful languages that are able to handle algorithms and other heavy tasks. Therefore, they use languages such as Python, Ruby, Java, and Javascript frameworks such as Node JS.
